| T-02-18 | Tampering (data integrity) | inline edit | mitigate | correctBrfField re-runs the deterministic scorer only and is auth-gated to the owner (Plan 04); corrected fields are labelled "Manuellt angiven" so a manual value is never confused with an extracted one (D-12) |
| T-02-19 | Information Disclosure | /sa-raknar-vi public page | mitigate | Page is static content + imported threshold constants only - no DB query, no user data, no Claude; renders the same numbers as the scorer |
| T-02-20 | Tampering | grade provenance shown to user | mitigate | The card displays the code-computed grade (D-08) and links/aligns to the public methodology so the grade is auditable, not a black box |
</threat_model>

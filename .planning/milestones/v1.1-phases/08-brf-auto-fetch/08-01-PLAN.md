## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| iXBRL/HTML document text → Claude extraction call | Untrusted external document content becomes the prompt payload (data, not instructions) |
| Server action → Supabase (analyses row) | Ownership-gated writes; RLS is the backstop |
| Migration DDL → live database | Schema change applied to production data |

Oversized-document bounding is handled at the fetch layer in Plan 02 (this plan only receives already-bounded text). |

Applies identically to the iXBRL branch. |

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → Postgres RPC | A browser-session-authenticated user invokes `claim_discovery_slice`; the RPC runs SECURITY DEFINER and must not let a user claim another user's job (scoped by `p_job_id` + upstream ownership check in Plan 03's action). |
| scraped payload → persisted row | Raw Booli Apollo entities / broker text (untrusted, PII-bearing) cross into `discovery_jobs.results` — only the allowlisted candidate shape may cross. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-01 | Elevation of Privilege | `claim_discovery_slice` SECURITY DEFINER | mitigate | `set search_path = public` pins the schema; function only mutates by `p_job_id`; `revoke all from public` + `grant execute to authenticated` — no anon access.

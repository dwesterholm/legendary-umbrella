## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| direct URL / devtools → /discover routes | An unauthenticated or non-owner request, or a request while the flag is off, must resolve to notFound() — no feature-existence or ownership leak. |
| persisted results → rendered card | Only allowlisted `DiscoveryCandidate` fields cross to the DOM; a Zod read-path guard rejects shape-drifted rows. |
| Vercel Cron → sweep route | The public cron endpoint must only reclaim genuinely-stale jobs via the atomic RPC, bounded per invocation. |
